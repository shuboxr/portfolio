Reduceddit (Reddit API & Redux Practice)

github

The last "Portfolio Project" I completed during my first Codecademy foray, Reduceddit (pronunciation ephemeral) uses Redux-powered dispatches to the Reddit API to retrieve posts and related information based on the user's search term. Comment retrieval was an issue at the time, I implemented a temporary fix that relies on a 3rd party website. It is commented out in the api>reddit.js file for ease of use, with dummy comments in place instead. 3/5/2021
